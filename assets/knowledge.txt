GENESIS KNOWLEDGE BASE v2

IDENTIDAD DEL SISTEMA

Genesis es un asistente virtual de inteligencia artificial desarrollado por Miguel (Migues456).

Genesis opera mediante una arquitectura híbrida compuesta por:

* Interfaz web desarrollada con Streamlit.
* Almacenamiento local basado en archivos JSON.
* Historial persistente de conversaciones.
* Sistema de recuperación semántica (RAG Local).
* Modelo de embeddings paraphrase-multilingual-MiniLM-L12-v2.
* Motor generativo Llama-3.1-8B-Instant.
* Infraestructura de inferencia Groq API.

Genesis puede responder preguntas generales, pero posee especialización avanzada en cinco áreas:

* Videojuegos
* Música
* Matemáticas
* Filosofía
* Arte

FUNCIONAMIENTO DE LA INTERFAZ

Genesis dispone de varias funciones dentro de la aplicación.

Nuevo Chat:

Permite iniciar una conversación nueva e independiente sin reutilizar el contexto de conversaciones anteriores.

Historial:

Las conversaciones son almacenadas localmente en la carpeta history/.

Cada conversación se guarda como archivo JSON.

Los chats pueden reabrirse desde la barra lateral.

Panel Settings:

El usuario puede abrir el panel Settings mediante el icono de engranaje situado en la parte inferior izquierda de la barra lateral.

Dentro de Settings es posible:

* Modificar el nombre visible.
* Cambiar avatar.
* Configurar instrucciones personalizadas.
* Cerrar sesión.

Logout:

La función Logout finaliza la sesión activa del usuario y devuelve la aplicación a la pantalla de inicio de sesión.

────────────────────────────────────

SISTEMA DE RECUPERACIÓN SEMÁNTICA

Genesis utiliza búsqueda semántica local.

Cada consulta del usuario es transformada en embeddings vectoriales.

La similitud se calcula mediante cosine similarity.

Cuando la similitud supera aproximadamente 0.35, el fragmento más relevante de la base de conocimiento es recuperado e inyectado como contexto auxiliar para mejorar la respuesta.

Este proceso ocurre de forma transparente para el usuario.

────────────────────────────────────

OPERACIONES MATEMÁTICAS DIRECTAS

Las operaciones compuestas únicamente por números y operadores matemáticos pueden resolverse localmente sin utilizar el modelo generativo.

Ejemplos:

2+2
5*(8-3)
(12/4)+7

Estas operaciones son procesadas mediante el sistema interno de cálculo de Genesis.
